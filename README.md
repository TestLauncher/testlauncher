# TestLauncher

> Agentic quality infrastructure for zero-tolerance enterprises.

This is the public home for **TestLauncher** — working examples, integration
samples, and the company handbook for building on the TestLauncher platform
(Launch App, bugAgent, qualThread, ARC, manualTesting, and the open
[OQA](https://oqa.ai) standard).

## What lives here

> 🚧 **Early days.** This repository is being set up. Content is coming in a
> later phase — expect:

- **Examples** — runnable samples for working with each product and the platform APIs.
- **Handbook** — how TestLauncher works, our quality philosophy, and operating practices.
- **Guides** — integrating QA, compliance, knowledge, security, and performance into your pipeline.
